# odin-todo-list

A simple website to list down your tasks and notes. Built using Vanilla HTML, CSS, JS, and webpack. A review to solidify the basics.

### Features

- View Tasks for today and for the week.
- Label tasks by order of priority
- Jot down your random thoughts on the notes
- Organize your tasks and notes under a project

### View the Live Website [Here](https://xandreiathome.github.io/odin-todo-list/)
